Shortcuts

Main window

Shortcut Action
F1 Open Help/About and Settings
Esc Minimize or restore the main window
Enter Add the task in the task box
Delete Remove the selected task
Up / Down Move through the task list
Ctrl+A Focus the task box
Ctrl+S Open Summary
Ctrl+D Open Summary Calculator
Ctrl+. Open the TinyTracker data folder
Ctrl+L Write a zero-minute label into today's CSV file
Ctrl+N Open a note window
Ctrl+V Paste into the task box, even when it is not focused
Ctrl+C Copy the selected task name when the list is focused or the window is minimized
Ctrl+0 to Ctrl+9 Quick-select a task by list position
Ctrl++ Move 1 minute from Nothing to the selected task
Ctrl+Shift++ Move 5 minutes from Nothing to the selected task
Ctrl+- Move 1 minute from the selected task back to Nothing
Ctrl+Shift+- Move 5 minutes from the selected task back to Nothing
F2 to F12 Run a custom function, if one is configured

If there is text in the task box, Ctrl++ can also add that task first.

Summary

Shortcut Action
Ctrl+P Previous day
Ctrl+N Next day
Ctrl+T Today
Ctrl+L Run the current Last X days view
Ctrl+< Move backward by the Last X days value
Ctrl+> Move forward by the Last X days value
Ctrl+F Focus the search box
Ctrl+D Open Summary Calculator
Ctrl+R Reset the running total

Summary Calculator

Shortcut Action
Esc Close the calculator

Note window

Shortcut Action
Esc Leave edit mode
Ctrl+Q Close the note when it is not in edit mode
